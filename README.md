## 📖 About The Project

This repository contains the source code for my personal portfolio website. It is designed to be a central hub for my professional identity, featuring a **responsive design**, **interactive project showcases**, and a **dedicated blog** for my game development insights.

### ✨ Key Features
* **Sector 11 Showcase:** A dedicated landing page for my FPS Battle Royale game inspired by the Bangladesh Liberation War.
* **Dynamic Components:** Uses vanilla JavaScript to load reusable components (like the Navbar) dynamically.
* **Custom 404 Page:** A unique, personality-driven "Lost Zoro" error page.
* **Contact Integration:** Fully functional contact form powered by [Formspree](https://formspree.io).
* **Responsive UI:** Optimized for all devices (Mobile, Tablet, Desktop) using custom CSS3 media queries.
